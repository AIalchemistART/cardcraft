‚ö†Ô∏è Important: Your Card Needs to Be Hosted Online

To use "Add to Home Screen" with the custom CardCraft icon, your business card HTML file must be hosted on a web server (not opened as a local file).

ÔøΩ Step 1: Host Your Card (Choose One)

Option A: Netlify Drop (Easiest - 30 seconds)

  1. Go to app.netlify.com/drop
  2. Drag your downloaded HTML file into the box
  3. Get instant URL like https://your-card.netlify.app

Option B: GitHub Pages (Free Forever)

  1. Create a GitHub account at github.com
  2. Create new repository, name it business-card
  3. Upload your HTML file and rename it to index.html
  4. Go to Settings ‚Üí Pages ‚Üí Enable GitHub Pages
  5. Your URL: https://username.github.io/business-card

üì± Step 2: Add to Your Phone's Home Screen

For iPhone/iPad (iOS):

  1. Open your hosted card URL in Safari
  2. Tap the Share button (square with arrow)
  3. Tap "Add to Home Screen"
  4. The CardCraft icon will appear - tap "Add"

For Android:

  1. Open your hosted card URL in Chrome
  2. Tap the menu (three dots)
  3. Tap "Add to Home screen"
  4. The CardCraft icon will appear - tap "Add"

üí° Pro Tips:

  • Share the URL: Send your hosted URL instead of the HTML file for easier sharing
  • QR Code: Your card already has a QR code that links to wherever it's hosted
  • Custom Domain: Both Netlify and GitHub Pages support custom domains for a professional touch
